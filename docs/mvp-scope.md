# MVP Scope

Included

- Authentication
- Guest Login
- Google Login
- Create Room
- Join Room
- QR Code
- Realtime Participants
- Turn System
- Timer
- Conversation Cards

Excluded

- AI
- Voice Recognition
- Chat
- Video Calls
- Gamification
- Rankings
- Notifications
- Premium Features
- Admin Panel
- Multi-language UI

Rule

If a feature is not listed as Included, it must not be implemented during the MVP.
