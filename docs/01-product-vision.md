# Let's Talk

## Vision

Let's Talk is a mobile-first Progressive Web Application (PWA) designed to facilitate in-person language conversation sessions.

Unlike traditional language learning platforms, Let's Talk does not focus on teaching grammar or vocabulary. Instead, it provides a structured environment where people can practice speaking naturally through guided conversations.

The application acts as a digital facilitator by organizing speaking turns, assigning conversation cards, and keeping every participant engaged.

The first supported language will be English, but the platform must be language-agnostic so new languages can be added without modifying the application architecture.

---

## Problem

Conversation groups often fail because:

- People don't know what to talk about.
- Some participants dominate the conversation.
- Beginners feel nervous.
- Sessions become repetitive.
- There is no structured flow.

Let's Talk solves these problems by providing guided conversation sessions.

---

## Target Users

- Friends learning languages
- Universities
- English clubs
- Teachers
- Companies
- Language communities

---

## MVP Goal

Validate that structured conversation sessions improve participation and engagement.

No AI features are included in the MVP.
